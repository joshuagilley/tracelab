# TraceLab Docs

This folder holds project documentation beyond local setup.

## Guides

- `ARCHITECTURE.md` - stack, folder structure, and system flow.
- `NEW_CONCEPT.md` - end-to-end process for adding a new concept.
- `NEW_LAB.md` - process for adding a brand-new lab section.
- `MONGO.md` - collection-level schema notes and common field usage.

## Source of truth

- Local development setup stays in repo-root `README.md`.
- Cursor workflow guidance for concept creation is in `.cursor/rules/new-concept.mdc`.
